# 🌱 Core Spring Framework

A complete learning repository for the **Spring Framework (Core)** created using **Java** and **Maven**. This repository contains beginner to intermediate examples covering Spring Core concepts such as Dependency Injection (DI), Inversion of Control (IoC), Bean Management, Bean Lifecycle, Autowiring, Annotation Configuration, Java Configuration, and more.

---

## 📌 About

This project is designed for learning and practicing **Spring Core Framework** concepts with clean code and proper project structure.

It includes practical examples demonstrating how Spring manages objects, injects dependencies, and simplifies Java application development.

## ⚙️ Clone Repository

```bash
git clone https://github.com/MdMohsinHaider/core-spring-framework.git
```

Move into the project directory

```bash

```

## ▶️ Run the Project

### Using Maven

